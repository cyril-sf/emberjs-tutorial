# emberjs-tutorial
Learn Ember.js

## Installation
```bash
$ npm install -g grunt-cli
$ npm install
```

## Running / Development

* `npm start`
* Visit your app at [http://localhost:8000](http://localhost:8000).

### Running Tests

* `npm start`
* Visit your tests at [http://localhost:8000/tests](http://localhost:8000/tests).
